# 🏥 Intelligent Health Information System (iHIS)

A comprehensive, production-ready healthcare information system with Flask REST API backend and vanilla JavaScript frontend.

## ✨ Features

### Multi-Role Support
- **Patients** - Medical records, appointments, prescriptions, lab results
- **Doctors** (20+ specialties) - EMR, SOAP notes, ICD-10 coding
- **Laboratories** - Test orders, results, quality control
- **Radiology** - Imaging requests, DICOM-ready, reports
- **Pharmacists** - Prescription verification, inventory
- **Nurses** - Vital signs, medication administration, care plans
- **Dentists** - Dental charts, imaging, procedures
- **Physiotherapists** - Assessments, exercises, progress tracking
- **Receptionists** - Appointments, registration, queue management
- **Administrators** - Staff management, reports, KPIs

### Modules
- 📋 EMR/EHR System
- 🔬 Laboratory Information System
- 🩻 Radiology Information System
- 💊 Pharmacy Management
- 🦷 Dental Module
- 🏋️ Physiotherapy Module
- 🤖 AI-Ready Architecture

### Security
- Role-Based Access Control (RBAC)
- JWT Authentication
- Password Hashing (bcrypt)
- Audit Logging

## 🛠️ Tech Stack

### Backend (REST API)
- **Framework:** Flask 3.0
- **Database:** SQLite (migratable to PostgreSQL/MySQL)
- **ORM:** SQLAlchemy 2.0
- **Authentication:** Flask-JWT-Extended
- **Migrations:** Flask-Migrate
- **CORS:** Flask-CORS

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Bootstrap 5
- **JavaScript** - Vanilla JS, Fetch API
